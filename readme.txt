Use emblending option for music in Blip.FM

== Description ==

Easy to use: [blipfm URL-to-music] or [blipfm ID-to-music]

This plugin was made to make easy the Blip.fm emblending option. This plugin is maintained by <a href="http://swasthya.marcocarvalho.com/">Marco Carvalho</a>. Visit the plugin home page: http://filosofiadevida.marcocarvalho.com/blip-fm-wordpress-plugin/

== Installation ==

The easy three steps:

1. Upload `blipfm/blipfm.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [blipfm URL-to-Music] or [blipfm ID-to-music] in your posts/pages.
